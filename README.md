quick set up for a CMake project on FedoraLinux-42.
This contains:
vcpkg
clang++
CMake.

the ts/exe/ directories contains the .sh files allow to config && build a CMake project

Usage:

clone the file at the ~/ dir

then chmod +x ~/quickCMake/q.sh

and just lanch ~/quickCMake/q.sh anywhere that you want it to build the setup
